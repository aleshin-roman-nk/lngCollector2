Create a new terrain
POST /api/Terrain
Create a new terrain.
Request data:
{
    "id": 0,
    "name": "Village",
    "description": "of nouns"
}
Response content:
{
    "id": 128,
    "name": "Village",
    "description": "of nouns"
}
